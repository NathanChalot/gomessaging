# gomessaging
Simple messaging Server/Client using socket from gorilla

### Requirement
```
go get github.com/gorilla/websocket
go get github.com/tsenart/vegeta
```

### Usage

To run server:
```
go run src/main.go
```
To run test:
```
go run src/main.go &
go run src/vegeta.go
```
